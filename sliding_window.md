## Partition and Removal

### Move Zeros

**Parition the array into two parts, the left subarray are non zeros, the right subarray are zeros**

example: `[1, 0, 0, 2, 3, 0, 4]` --> `[1, 2, 3, 4, 0, 0, 0]`
